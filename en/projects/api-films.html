APIfilms - REST API for Films

This page presents the case study for the APIfilms project, including a project overview and the tools used.

Project Link

Project Overview

APIfilms is a REST API developed with ASP.NET Core and C# for film management. The API allows creating, reading, updating, and deleting film information, providing a standardized interface to access cinematographic data.

This API demonstrates mastery of backend development with ASP.NET Core, the implementation of a RESTful architecture, HTTP request handling (GET, POST, PUT, DELETE), JSON serialization, and best practices for modern web API development.

Among the key features, we find complete CRUD operations on films, appropriate HTTP error handling, data validation, API documentation, and a modular architecture facilitating maintenance and extension.

The system is designed to be scalable, maintainable, and compliant with REST standards, enabling easy integration with frontend applications or other services.
