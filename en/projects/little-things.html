Project Overview

Little Things is a collection of Python utilities developed to automate daily tasks and improve productivity. The project groups several utility scripts including a wallpaper changer, a task manager, and a file renaming tool.

This collection demonstrates mastery of Python for automation, file and system manipulation, operating system interaction, and the creation of practical tools to improve daily efficiency.

Among the key features, we find an automatic wallpaper changer, a command-line task manager, a bulk file renaming tool, and other system utilities for file and process management.

The system is designed to be modular and easily extensible, allowing the addition of new utilities as needed, while demonstrating a practical approach to Python programming.
