Project Overview

DortanApp is a WPF-based software that incorporates a database, designed for recording activities within a town hall. This project was developed as part of a university assignment.

The application enables town hall personnel to create or select activities and input the reservation date along with other pertinent details. It aims to streamline activity management and enhance operational efficiency in the town hall. It boasts an intuitive user interface and robust data management capabilities to support event tracking and planning.

Key features of DortanApp include reservation management with an intuitive graphical interface and filtering options for easier navigation.
