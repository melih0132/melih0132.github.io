Project Overview

The Quiz JSON application is a quiz application developed with Python that uses JSON files to store questions and answers. The application allows users to answer quiz questions and obtain their scores.

This application demonstrates mastery of JSON data manipulation with Python, command-line user interface creation, file management, and the implementation of business logic for interactive quizzes.

Among the key features, we find loading questions from JSON, answer management and validation, score calculation, management of multiple quiz categories, and an interactive command-line user interface.

The system is designed to be extensible and easily modifiable, allowing the addition of new questions simply by modifying JSON files, without changing the code.
