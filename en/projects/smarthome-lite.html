SmartHome Lite

Distributed smart-home portfolio: web and mobile control of lights and sockets, Z-Wave presence sensing, IF-THEN rules, ambient-based dimming and ML on the backend. Main repository: smarthome-lite.

Main repository

Overview

The big-projects/smarthome-lite folder in all-my-projects redirects to the dedicated repo. The stack includes PostgreSQL (relational and time-series data), FastAPI (SQLAlchemy, JWT), Next.js (App Router, TypeScript, Tailwind), a Kotlin / Jetpack Compose Android app, and a Raspberry Pi edge with a Flask API and Z-Wave JS UI.

Features

Control and automation

Manual control of lights and sockets, a rule engine and predefined scenarios.

Sensors and history

Presence (e.g. MultiSensor 7), history and dashboards for presence and luminosity.

Machine learning

Scenario classification and intensity prediction with ML models (pandas, NumPy).
