Project Overview

The Weather App is a cross-platform mobile application developed with Flutter that allows users to view real-time weather information for any city in the world. The application uses the OpenWeatherMap API to retrieve accurate and up-to-date weather data.

This application demonstrates the integration of external REST APIs in a Flutter application, asynchronous state management, network error handling, and the creation of an intuitive user interface to display complex weather data in a clear and accessible manner.

Among the key features, we find city search by name, display of current weather conditions (temperature, humidity, wind speed, atmospheric pressure), display of weather forecasts for several days, geolocation management to automatically get the weather for the current location, and a responsive user interface adapted to different screen sizes.

The system is designed to provide a smooth user experience with elegant animations, optimized API request management to minimize data consumption, and robust error handling to ensure application stability even in case of network issues.
