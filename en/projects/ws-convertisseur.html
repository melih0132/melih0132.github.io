Project Overview

WSConvertisseur is a web service developed with ASP.NET Core and C# that provides currency and unit conversion functionalities. The service exposes a REST API allowing conversion of different units of measurement and currencies in a standardized manner.

This web service demonstrates mastery of SOAP/REST web service development with ASP.NET Core, the implementation of conversion business logic, HTTP request handling, data serialization, and best practices for developing reusable web services.

Among the key features, we find currency conversion with exchange rates, unit conversion (length, weight, temperature, etc.), exposure via REST API, error handling and validations, and a modular architecture facilitating the addition of new conversion types.

The system is designed to be scalable, maintainable, and easily integrable into other applications, providing a centralized and standardized conversion service.
