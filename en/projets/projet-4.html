Project Overview

This project focuses on developing a comprehensive database for managing bird count data, empowering the League for the Protection of Birds (LPO) to efficiently collect, analyze, and present vital information. Built on PostgreSQL, the database seamlessly integrates with tools like Excel and Power BI to enhance data analysis and visualization capabilities.

The database modeling phase included creating an Entity-Relationship Diagram (ERD) to define key entities such as bird counts, observers, species, and sites, along with their relationships and constraints. This was followed by the Logical Data Model (LDM), which translated the ERD into relational tables specifically tailored for PostgreSQL.

In the database setup, tables were created with defined constraints, including primary and foreign keys, to ensure data integrity. Simulated bird count records were then populated into the database, and SQL queries were developed for efficient data management and analysis.

For statistical analysis, relevant data was extracted for use in Excel, where descriptive statistics were performed to assess the number of individuals per species, geographic distribution of observations, and population trends over time. Visualizations were created to represent these findings effectively.

The project culminated in the development of an interactive report using Power BI. Consolidated data was imported to design a dynamic dashboard that displays trends in bird populations, identifies areas of high biodiversity, and facilitates annual or seasonal comparisons.
