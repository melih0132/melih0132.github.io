ADMINISTER - Level 1

Critical Learning AC13 : Simple system administration

Back to skills
Level definition

Level 1 (AC13) of the ADMINISTER competency corresponds to simple system administration. At this level, I am able to configure basic servers, manage users and permissions, and perform backups.

This level represents my first steps in system administration, where I learned the fundamental concepts of server management and service configuration.

Critical Learnings for Level 1

• AC13.01 | Install and configure a development workstation : Development environment configuration, tool installation, basic settings
• AC13.02 | Install necessary software : Software installation, dependency management, tool configuration
• AC13.03 | Configure a simple work environment : Basic environment setup, initial configuration, simple settings

Resources studied in class (S1-S2)

R1.03 : Introduction to computer architecture
Understanding hardware architecture, computer components, basic functioning.

R1.04 : Introduction to operating systems and their functioning
Operating system principles, process management, files and permissions, basic administration.

Topics covered

• Installation and configuration of a development workstation: IDE, tools, environment
• Software installation: dependency management, configuration
• Simple work environment configuration: environment variables, paths

Reference knowledge

• Hardware architecture: CPU, memory, storage, peripherals
• Operating systems: processes, threads, files, permissions
• Process management: creation, synchronization, communication
• Files and permissions: access rights, basic security

Content to complete

To be completed with your personal projects and experiences:

Backups
Traces and evidence

Configurations : Configuration files, installation scripts, and parameter documentation.

Backups : Backup scripts, documented strategies, and restoration procedures.

Pedagogical reflection

Level 1 allowed me to acquire the basics of system administration. I learned to configure servers, manage users, and implement basic backup strategies.
