ADMINISTER - Level 1

Critical Learning AC13 : Simple system administration

Back to skills
Level definition

Level 1 (AC13) of the ADMINISTER competency corresponds to simple system administration. At this level, I am able to configure basic servers, manage users and permissions, and perform backups.

This level represents my first steps in system administration, where I learned the fundamental concepts of server management and service configuration.

Critical Learnings for Level 1

• AC13.01 | Install and configure a development workstation : Development environment configuration, tool installation, basic settings
• AC13.02 | Install necessary software : Software installation, dependency management, tool configuration
• AC13.03 | Configure a simple work environment : Basic environment setup, initial configuration, simple settings

Reference situations

Development Workstation Configuration
Installation and configuration of development workstations for all projects (Heva, Uber Laravel, Uber Vue.js/.NET, APIfilms, etc.). Installation of necessary software (IDEs, development tools, frameworks), and configuration of simple work environments for local development.

Academic Projects - Work Environments
Configuration of development environments for university projects (DortanApp, Evit'Shuriken, Weather App, etc.). Installation of necessary tools (Visual Studio, VS Code, Flutter, etc.) and basic environment configuration.

little-things - Linux System Utilities
Installation and configuration of Linux system tools, file utility management, and work environment configuration for automation.
