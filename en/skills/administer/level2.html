Level definition

Level 2 (AC23) of the ADMINISTER competency corresponds to complex system administration. At this level, I am able to deploy applications with Docker, configure CI/CD environments, and monitor performance.

This level represents an evolution in my ability to manage more complex infrastructures, automate deployments, and ensure system monitoring.

Critical Learnings for Level 2

• AC23.01 | Deploy services in a network architecture : Service deployment, network configuration, distributed architecture
• AC23.02 | Use servers and virtualization tools : Virtualization, containers, orchestration, server management
• AC23.03 | Secure services and data : Service security, data protection, security management
