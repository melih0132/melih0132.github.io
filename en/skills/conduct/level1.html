CONDUCT - Level 1

Critical Learning AC15 : Simple project management

Back to skills
Level definition

Level 1 (AC15) of the CONDUCT competency corresponds to simple project management. At this level, I am able to plan project tasks, use basic management tools, and track work progress.

This level represents my first steps in project management, where I learned the fundamental concepts of planning and project tracking.

Critical Learnings for Level 1

• AC15.01 | Identify business needs of clients and users : Needs analysis, understanding business requirements, identifying objectives
• AC15.02 | Set up project management tools : Tool selection and configuration, project management process setup
• AC15.03 | Identify stakeholders and project phases : Stakeholder identification, project phase definition, organization

Resources studied in class (S1-S2)

R1.08 : Introduction to organizational management
Understanding organizations, structures, organizational processes.

R1.12 : Personal and professional project
Personal and professional development, reflection on professional project.

SAÉ 1.05 : Requirements gathering
Requirements gathering techniques, user needs identification.

Topics covered

• Identification of business needs of clients and users: interviews, questionnaires
• Setup of project management tools: Trello, GitHub Projects, basic tools
• Identification of stakeholders and project phases: stakeholders, lifecycle

Reference knowledge

• Requirements gathering methods: interviews, questionnaires, observation
• Basic project management tools: planning, monitoring, communication
• Project lifecycle: phases, milestones, deliverables
• Project stakeholders: client, contractor, users

Reference Situations (Academic Projects)
