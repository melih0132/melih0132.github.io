MANAGE - Level 1

Critical Learning AC14 : Simple data management

Back to skills
Level definition

Level 1 (AC14) of the MANAGE competency corresponds to simple data management. At this level, I am able to design relational databases, use SQL to manipulate data, and manage data integrity.

This level represents my first steps in data management, where I learned the fundamental concepts of relational databases and the SQL language.

Critical Learnings for Level 1

• AC14.01 | Design and implement a relational database : Database modeling, relational design, schema implementation
• AC14.02 | Query and update a database : SQL queries, data manipulation, database updates

Reference situations

LPO - PostgreSQL Database
Design and implementation of a PostgreSQL relational database for tracking bird populations. MCD/MLD modeling, table and relationship creation, and SQL query development to query and update the database.

DortanApp - PostgreSQL Database
Design and implementation of a PostgreSQL relational database for municipal reservation management. Database schema creation, relationship definition, and query development to query and update reservations.

HEVA - PostgreSQL Database
Design and implementation of a PostgreSQL relational database for event management. Modeling of relationships between users, events and reservations, and query development to query and update data.

Ranking - SQLite Database
Design and implementation of a SQLite relational database for the ranking system. Schema creation, table definition, and query development to query and update rankings.

Pedagogical reflection

Level 1 allowed me to acquire the basics of data management. Through projects like LPO, I learned to model databases, use SQL, and ensure data integrity.
