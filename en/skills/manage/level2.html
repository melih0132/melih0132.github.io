MANAGE - Level 2

Critical Learning AC24 : Complex data management

Back to skills
Level definition

Level 2 (AC24) of the MANAGE competency corresponds to complex data management. At this level, I am able to optimize query performance, use analysis tools like Power BI, and manage large volumes of data.

This level represents an evolution in my ability to analyze and visualize data, optimize performance, and manage larger data volumes.

Critical Learnings for Level 2

• AC24.01 | Optimize database performance : Performance optimization, indexing, optimized queries
• AC24.02 | Ensure data security and integrity : Data security, referential integrity, data protection
• AC24.03 | Organize data restitution and visualization : Data restitution, visualization, reports and dashboards

Resources studied in class (S3-S4)

R2.06 : Database exploitation
Performance optimization, data security and integrity, data visualization and reporting.

R3.07 : SQL in a programming language
SQL integration in applications, ORM, data management in code.

Topics covered

• Database performance optimization: indexing, optimized queries
• Data security and integrity: authentication, validation, transactions
• Data visualization and reporting: reports, dashboards, analytics

Reference knowledge

• Query optimization: execution plan, indexes, statistics
• Indexing: B-tree, hash index, composite indexes
• Transactions: ACID, isolation, rollback
• Views and stored procedures: abstraction, reusability
• ORM: object-relational mapping, ORM queries
• Analytics: reports, visualization, business intelligence

Reference Situations (Academic Projects)

• SAÉ 2.04 : Dataset, Complex Queries, PowerBI - Database performance optimization (R2.06), data security and integrity, data restitution and visualization with PowerBI. I used resources R3.07 (Triggers, Procedures, NoSQL) to optimize and secure the database.

• SAÉ 3.01 : Clone Uber (Laravel) - Database performance optimization, use of NoSQL databases (MongoDB) via R4.03, data security and integrity in a complete web application.

• SAÉ 4.01 : Clone Uber (.NET + Vue.js) - Management of complex data in a full-stack application, performance optimization, data security and integrity, use of relational and NoSQL databases.
