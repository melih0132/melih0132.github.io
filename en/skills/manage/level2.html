MANAGE - Level 2

Critical Learning AC24 : Complex data management

Back to skills
Level definition

Level 2 (AC24) of the MANAGE competency corresponds to complex data management. At this level, I am able to optimize query performance, use analysis tools like Power BI, and manage large volumes of data.

This level represents an evolution in my ability to analyze and visualize data, optimize performance, and manage larger data volumes.

Critical Learnings for Level 2

• AC24.01 | Optimize database performance : Performance optimization, indexing, optimized queries
• AC24.02 | Ensure security and integrity : Data security, referential integrity, data protection
• AC24.03 | Organize data restitution and visualization : Data restitution, visualization, reports and dashboards

Reference situations

LPO - PowerBI and PostgreSQL Optimization
Optimization of PostgreSQL database performance with strategic indexing, ensuring data security and integrity, and organizing data restitution and visualization with PowerBI to create interactive reports and dashboards.

HEVA - Optimization and Analytics
Optimization of PostgreSQL database performance, ensuring data security and integrity (authentication, validation), and organizing data restitution with analytics for usage analysis.

Uber Laravel - PowerBI and Optimization
Optimization of PostgreSQL database performance for transport data, ensuring data security and integrity, and organizing restitution with PowerBI for analytics.

DocumentApp - iOS Document Management
Organization of document management on iOS, ensuring file security and integrity, and performance optimization for document manipulation.
