OPTIMIZE - Level 1

Critical Learning AC12 : Simple algorithm optimization

Back to skills
Level definition

Level 1 (AC12) of the OPTIMIZE competency corresponds to simple algorithm optimization. At this level, I understand algorithmic complexity and I am able to implement basic algorithms and use appropriate data structures.

This level represents my first steps in algorithm analysis and optimization, where I learned the fundamental concepts of algorithmic complexity and the importance of choosing data structures.

Critical Learnings for Level 1

• AC12.01 | Analyze a problem by breaking it down into sub-problems : Decomposition of complex problems, identification of sub-problems, modular approach
• AC12.02 | Build a simple algorithm to solve a given problem : Creation of basic algorithms, implementation of simple solutions, solving elementary algorithmic problems

Resources studied in class (S1-S2)

R1.01 : Introduction to development
Introduction to basic algorithms, control structures, and simple problem solving.

R1.06 : Discrete mathematics
Mathematical foundations for algorithmics: logic, sets, relations, functions.

R2.07 : Graphs
Introduction to data structures and graph algorithms.

Topics covered

• Problem breakdown into sub-problems: analysis, decomposition, modularity
• Construction of simple algorithms: sequential, conditional, iterative
• Basic data structures: lists, arrays, dictionaries
• Basic algorithmic complexity: execution time, memory space

Reference knowledge

• Elementary algorithms: sorting, searching, data manipulation
• Data structures: lists, arrays, trees, graphs
• Complexity analysis: Big O notation, time and space

Reference Situations (Academic Projects)

• SAÉ 1.01 : Evit'Shuriken - Development of a simple game, implementation of basic algorithms for gameplay management. I used resources R1.01 (C#) to break down problems into sub-problems and build simple algorithms.

• SAÉ 1.02 : Evit'Shuriken - Continuation of the project, algorithm optimization, performance improvement. This phase allowed me to analyze complexity and optimize the implemented algorithms.

• SAÉ 2.02 : Graph Modeling, Decision Support Tools, Poliotages, Excel - Use of graph theory (R2.07) for problem modeling, decision support tools. This project allowed me to apply discrete mathematics (R1.06) and graph theory to solve complex problems.
