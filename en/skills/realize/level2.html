REALIZE - Level 2

Critical Learning AC21 : Development of complex applications

Back to skills
Level definition

Level 2 (AC21) of the REALIZE competency corresponds to the development of complex applications. At this level, I am able to design application architecture, integrate multiple technologies and frameworks, and implement testing and deployment processes.

This level represents a significant evolution in my ability to manage technical complexity, integrate different technologies and implement automated processes to guarantee application quality and deployment.

Critical Learnings for Level 2

• AC21.01 | Develop and implement functional and non-functional specifications : Analysis and formalization of needs, implementation of functional specifications and technical constraints
• AC21.02 | Apply ergonomics and accessibility principles : Design of ergonomic interfaces, respect for accessibility standards, optimal user experience
• AC21.03 | Apply good design and programming practices : Design patterns, SOLID principles, clean and maintainable code
• AC21.04 | Verify and validate an application through testing : Unit tests, integration tests, validation tests, quality assurance

Reference situations

HEVA - Event Management Platform (FastAPI/React)
Development of a complete web application with FastAPI (Python) for the backend and React/TypeScript for the frontend. Development and implementation of functional and non-functional specifications, application of ergonomics and accessibility principles, good design and programming practices, and verification through automated tests.

Uber Vue.js/.NET - Transportation Application
Development of a full-stack application with Vue.js and .NET Core. Complex architecture, integration of multiple services (geolocation, payment), application of best practices, and implementation of integration tests.

Uber Laravel - PHP Web Application
Development of a complete web application with Laravel (PHP) and Vue.js. MVC architecture, formalized functional specifications, ergonomic interfaces, and validation through tests.

APIfilms - RESTful API ASP.NET Core
Development of a RESTful API for film management with ASP.NET Core. Implementation of functional specifications, good design practices, and validation through unit and integration tests.

Pokemon Trainer Manager - iOS Swift/UIKit Application
Development of an iOS mobile application with Swift and UIKit. Design of ergonomic user interfaces, application of iOS accessibility principles, and implementation of features according to specifications.

Pedagogical reflection

Level 2 was a turning point in my learning, allowing me to move from implementing isolated features to designing complete and integrated applications. Projects like UberS4 and HEVA particularly taught me to manage technical complexity, integrate different technologies and think in terms of architecture.

The main challenges were managing increasing complexity, integrating multiple technologies, and implementing automated processes. The solutions provided were adopting good architectural practices, using design patterns, and automating testing and deployment processes.
