Projets Découvrez ici une sélection de projets académiques et personnels que j'ai réalisés.

Evit'Shuriken

Evit'Shuriken est un jeu WPF que j'ai créé, combinant des connaissances en .NET/C# et en XAML. Ce projet a été réalisé durant ma première année en BUT Informatique.

DortanApp

DortanApp est une application développée en .NET/C# avec WPF, intégrant une base de données PostgreSQL pour la gestion des réservations dans une mairie.

KUBE

KUBE est un jeu réalisé via Unity en .NET/C#. Ce projet m'a permis de découvrir les outils de développement de jeux.

Phaser Games

Jeux développé avec Phaser, utilisant Express pour le backend, Socket.io pour la communication en temps réel, et Nodemon pour le rechargement automatique.

LPO

Un projet universitaire gérant le comptage des oiseaux pour la LPO, utilisant PostgreSQL pour la gestion, Excel pour l'analyse, et Power BI pour la visualisation.

Application type Uber

Une application de transport permettant la réservation de trajets en temps réel, l’évaluation des conducteurs et le suivi sur une carte interactive. Développée avec Laravel pour le backend, PostgreSQL pour la base de données, Node.js et Express pour les services en temps réel, et intégration de Stripe pour les paiements sécurisés.
