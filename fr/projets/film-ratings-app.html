Aperçu du projet

FilmRatingsApp est une application desktop moderne développée avec WinUI 3 et .NET Core pour la gestion et l'évaluation de films. L'application offre une interface utilisateur moderne et fluide utilisant les dernières technologies Microsoft pour les applications Windows.

Cette application démontre la maîtrise du développement d'applications desktop modernes avec WinUI 3, l'utilisation de XAML pour créer des interfaces utilisateur riches, l'intégration avec .NET Core pour la logique métier, et l'implémentation de patterns modernes de développement d'applications Windows.

Parmi les fonctionnalités clés, on retrouve la gestion de films avec création, modification et suppression, le système de notation et d'évaluation, l'affichage de listes avec filtrage et tri, une interface utilisateur moderne avec animations et transitions fluides, et la persistance des données.

Le système est conçu pour offrir une expérience utilisateur native Windows optimale, en respectant les guidelines de design Microsoft Fluent Design et en utilisant les composants WinUI 3 pour garantir une intégration parfaite avec Windows 10/11.
