Aperçu du projet

L'application Quiz JSON est une application de quiz développée avec Python qui utilise des fichiers JSON pour stocker les questions et réponses. L'application permet aux utilisateurs de répondre à des questions de quiz et d'obtenir leurs scores.

Cette application démontre la maîtrise de la manipulation de données JSON avec Python, la création d'interfaces utilisateur en ligne de commande, la gestion de fichiers, et l'implémentation de logique métier pour les quiz interactifs.

Parmi les fonctionnalités clés, on retrouve le chargement de questions depuis JSON, la gestion des réponses et validation, le calcul de scores, la gestion de plusieurs catégories de quiz, et une interface utilisateur interactive en ligne de commande.

Le système est conçu pour être extensible et facilement modifiable, permettant l'ajout de nouvelles questions simplement en modifiant les fichiers JSON, sans modifier le code.
