Aperçu du projet

WSConvertisseur est un service web développé avec ASP.NET Core et C# qui fournit des fonctionnalités de conversion de devises et d'unités. Le service expose une API REST permettant de convertir différentes unités de mesure et devises de manière standardisée.

Ce service web démontre la maîtrise du développement de services web SOAP/REST avec ASP.NET Core, l'implémentation de logique métier de conversion, la gestion des requêtes HTTP, la sérialisation de données, et les bonnes pratiques de développement de services web réutilisables.

Parmi les fonctionnalités clés, on retrouve la conversion de devises avec taux de change, la conversion d'unités (longueur, poids, température, etc.), l'exposition via API REST, la gestion des erreurs et validations, et une architecture modulaire facilitant l'ajout de nouveaux types de conversions.

Le système est conçu pour être scalable, maintenable et facilement intégrable dans d'autres applications, offrant un service de conversion centralisé et standardisé.
